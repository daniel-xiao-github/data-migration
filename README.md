## Usage

```bash
unzip rest2adb-js-1.0.0.zip -d rest2adb
cd rest2adb

# Linux / macOS
./rest2adb.sh web

# Windows
rest2adb.bat web
```

The launcher will check for Node.js (>= 18) and prompt to install it if missing.
It will also prompt to run `npm install` for dependencies on first run.
